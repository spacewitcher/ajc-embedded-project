\doxysubsection{Detailed Description}
Class for describing the actions that are to be performed by the PCA9685 controller. 

This class describes the different actions which are to be performed by the PCA9685 controller. It allows to control the position of the yellow and blue flags via servomotors. Certain actions are programmed after receiving specific data from the Slave Raspberry Pi and before sending data to the TCP server on which all the data monitored by the Slave is stored.
