\doxysubsection{Detailed Description}
Class for controlling the PCA9685 PWM chip. 

This class provides an interface for controlling the PCA9685 PWM chip using the \mbox{\hyperlink{class_i2_c}{I2C}} bus on a Raspberry Pi. It allows to change the positions of the servomotors based on the desired angle (in degrees).
